## 📋 系統概述

外送員系統採用雙模式設計：
1. **📦 接單模式** - 選擇和確認訂單
2. **🗺️ 導航模式** - 85%地圖導航配送

### 🔄 操作流程

#### **步驟1：選擇訂單**
- 瀏覽各地區可接訂單
- 點擊訂單卡片進行勾選
- 底部橫條實時顯示已選數量

#### **步驟2：確認接單**
點擊 `[確認接單 >]` 後，橫條向上展開：

```
┌─────────────────────────────────────────┐
│           🛒 已選訂單 (2筆)              │
├─────────────────────────────────────────┤
│ 📦 李小華 - 三峽區民權街 - NT$185      │  
│ 📦 張小美 - 樹林區樹新路 - NT$210      │
├─────────────────────────────────────────┤
│ 總計: 2筆訂單 | 總金額: NT$395         │
├─────────────────────────────────────────┤
│        [🔒 鎖定訂單(30秒)]              │ ← 三階段按鈕
└─────────────────────────────────────────┘
```

#### **步驟3：三階段確認流程**

**階段1：鎖定訂單**
```
│        [🔒 鎖定訂單(30秒)]              │
```
- 防止其他外送員搶單
- 30秒保護時間
- 訂單卡片顯示鎖定狀態

**階段2：優化路線**  
```
│        [🧭 優化配送路線]               │
```
- Google Maps API計算最佳順序
- 顯示路線優化結果

**階段3：開始配送**
```
│        [🚀 開始配送]                   │
```
- 切換到導航模式
- 開始第一筆訂單配送

```
├─────────────────────────────────────────┤
│ 🎛️ 狀態控制列 (15%)                   │
│ 📦 2筆訂單 ⏰ 10:30  👤 李小華 [詳情] │ 
│ 📞 聯絡客戶 📷 拍照 🚨 問題回報 [完成] │
└─────────────────────────────────────────┘
```

### 🎛️ 狀態控制列功能

#### **第一排：訂單資訊**
```
📦 2筆訂單  ⏰ 10:30  👤 李小華  [詳情➤]
```
- **📦 2筆訂單**: 當前批次剩余訂單數
- **⏰ 10:30**: 預計送達時間
- **👤 李小華**: 當前目標客戶
- **[詳情➤]**: 展開客戶訂單詳情

#### **第二排：操作按鈕**
```
📞 聯絡客戶  📷 拍照送達  🚨 問題回報  [✅完成配送]
```
- **📞 聯絡客戶**: 一鍵撥打電話
- **📷 拍照送達**: 拍照並自動發LINE給客戶  
- **🚨 問題回報**: 回報配送問題
- **[✅完成配送]**: 完成當前訂單
