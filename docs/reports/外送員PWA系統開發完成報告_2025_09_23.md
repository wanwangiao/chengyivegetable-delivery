#### 2. 核心交互功能

✅ **接單流程 (購物車概念)**:
1. 瀏覽可接訂單 (狀態=`packed`)
2. 勾選想要的訂單
3. 訂單進入底部"已選訂單欄"
4. 鎖定訂單防止被搶
5. 點擊"開始配送"確認接單

✅ **配送執行流程**:
1. 自動切換到85%地圖導航模式
2. 點擊狀態列展開訂單詳情
3. **必須拍照才能完成配送**
4. 拍照後自動LINE通知客戶
5. 完成後自動導航下一筆訂單

✅ **問題回報系統**:
- 連絡不到客戶
- 地址有誤
- 其他問題

#### 3. PWA技術特色

✅ **Progressive Web App支援**:
