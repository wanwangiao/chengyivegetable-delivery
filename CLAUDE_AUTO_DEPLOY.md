# 🤖 Claude Code 自動部署觸發系統

## ✨ 魔法關鍵字

當您在對話中說出以下任何關鍵字，我會自動執行部署：

### 🚀 部署觸發關鍵字：
- **"請更新進度記錄並推送部署"** ⭐ (推薦)
- "更新進度並部署"
- "推送部署"  
- "部署到線上"
- "更新線上版本"
- "同步到生產環境"
- "deploy"
- "update and deploy"

## 🎯 使用方法

### 方法1：對話觸發 (最簡單)
您只需要在對話中說：
> "請更新進度記錄並推送部署"

我會自動執行：
1. 📋 更新 CLAUDE.md 進度記錄
2. 📦 提交所有更改
3. 📤 推送到 GitHub
4. 🚀 部署到 Vercel 生產環境

### 方法2：命令觸發
```bash
npm run auto-deploy
```

### 方法3：手動觸發
```bash
npm run claude-deploy "請更新進度記錄並推送部署"
```

## 🧠 智能功能

✅ **自動檢測更改**：有更改才提交
✅ **智能進度更新**：自動更新 CLAUDE.md 時間戳
✅ **完整部署流程**：提交 → 推送 → 部署
✅ **錯誤處理**：失敗時提供手動指引
✅ **狀態回饋**：每個步驟都有清楚的進度顯示
